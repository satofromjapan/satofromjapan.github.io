About

My name is Masato Takahashi, and I'm a web developer with a background in project management.

Before becoming a web developer, I had many opportunities to work with web developers who were creating exciting new things which I had always admired. This inspired me to learn about the technologies that power web applications and coding, which eventually lead to my decision to change my career path and pursue being a web developer.

I completed an immersive coding bootcamp at Coding Dojo in San Jose, where I learned 3 full stacks including: Python, MEAN, and Ruby with over 1000 hours of coding experience. I have also gained working knowledge of many other frameworks and web development processes. Currently, my favorite technology is Ruby on Rails.

I look forward to learning new technologies and build more complex web applications!
